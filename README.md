# 몽상의 서

Book of Reverie is a static one-page prototype for a dreamlike card board game.

## Prototype Scope

- Claim a daily reward once.
- Pick an opponent from the dream board.
- See the opponent choose a card first.
- Select an available response card.
- Start a battle, lose, and discover rule fragments.

## Local Preview

Open `index.html` in a browser, or deploy the folder as a static site on Vercel.
